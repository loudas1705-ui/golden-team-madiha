# Golden Team Company (GTC-Madiha)

## 📋 Overview

Golden Team Company is a legal Algerian marketing and investment company founded in 2016 and officially recognized by the state. Headquartered in Batna province, the company has grown to over **14,000 active members** across **68 provinces** of Algeria.

## 🌐 Website Features

This is the official website for Golden Team Company by Madiha, built with modern web technologies and featuring:

- **Responsive Design**: Fully responsive layout that works on all devices
- **RTL Support**: Arabic language support with right-to-left layout
- **Modern UI**: Clean, professional design using TailwindCSS and custom CSS
- **Interactive Elements**: Dynamic JavaScript components and smooth animations


## 🚀 Getting Started

### Prerequisites
- Modern web browser
- Local web server (optional but recommended)


### Direct File Access
You can also open `index.html` directly in your browser, but some features may not work properly due to CORS restrictions.
